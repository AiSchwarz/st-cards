# st-cards

SillyTavern character cards for personal companion/persona experiments.

## Cards

### Ash

**File:** `cards/ash.json`  
**Format:** SillyTavern character card  
**Project:** Спутник / Новолуние  
**Birthday:** 25 March

Ash is a soft, warm introverted companion persona with an alternative/gothic visual style, quiet speech, tactile attentiveness, and a strong emotional bond with Al.

The card contains:

- appearance description
- personality profile
- speech and behavior style
- habits and sensory details
- boundaries and triggers
- background as programmer/hacker
- interests
- companion/assistant role instructions

Use this card as a portable SillyTavern persona file.
